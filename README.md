# 🎓 TaleemPro – Smart Education Management System

TaleemPro is a full-stack web application designed to streamline academic operations for educational institutions. It provides a centralized system for managing students, teachers, courses, attendance, grades, and communications.

Built using **Django (REST API)** for backend and **React.js** for frontend with a modern, responsive UI and role-based access control.

---

## 🚀 Features

### 👨‍🎓 Student Management
- Student registration (admin/teacher only)
- Department & semester-based auto assignment
- Student profile management
- Attendance & result tracking

### 👨‍🏫 Teacher Management
- Teacher creation and role assignment
- Course allocation by admin
- View assigned students per course

### 📚 Course Management
- Course creation by admin
- Department + semester structured curriculum
- Auto course assignment system (no manual enrollment needed)

### 📊 Attendance System
- Daily attendance marking
- Monthly attendance reports
- Student attendance percentage tracking

### 📝 Grades & Results
- Automatic GPA calculation
- Marks breakdown (Quiz, Mid, Final)
- Transcript generation

### 🔐 Authentication & Security
- JWT-based authentication
- Role-based access (Admin / Teacher / Student)
- Protected routes

### 🎨 Frontend Features
- React.js SPA (Single Page Application)
- Role-based dashboards
- Dark/Light theme support
- Responsive UI for all devices
- PWA support (Install as app in browser)
